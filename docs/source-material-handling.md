# Source Material Handling

This project was completed as a guided data engineering project. The original course exercise and interview-prep documents are kept locally under `notes/private/course-source-docs/`, which is ignored by Git.

The public repo does not need to include those original documents. Instead, the `learning-notes/` folder contains rewritten explanations in my own words based on the project implementation and the concepts reinforced by the exercises.

This keeps the repository professional while still making it useful as a future study reference.
